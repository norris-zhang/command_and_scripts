- ### node version manager switch versions
```shell
$ nvm ls
$ nvm use 16.14.0
```
- ### list remote available versions
```shell
$ nvm ls-remote --lts
$ nvm version-remote --lts
```
- ### install a version
```shell
$ nvm install node # "node" is an alias for the latest version
$ nvm install 14.7.0
```
